# Eve Navigator - 跨星系导航插件

基于 EVE 宇宙图谱的 AstrBot 插件，通过远程导航 API 计算最优跳跃路线。

## 功能

- 跨星系导航路线规划（支持多种安全模式：super / capital / none）
- 三神裔黑隐导航（自动选择最近起点）
- 星系名称本地模糊匹配（前缀 / 子串，大小写不敏感）
- 高安星系终点拦截（sec > 0.5）
- 完整的导航输出：跳跃序列、星门/跳桥/跳跃方式、安等、落点设施

## 数据源

- 导航路线：远程 API（`NAV_API_BASE_URL`，默认 `aio.dusy.run`）
- 星系索引：本地 `data/universe_map.json`，用于模糊匹配和安全等级查询

## 命令

### `/nav` — 跨星系导航

```
/nav <起点星系> <终点星系> [安全模式] [跳跃距离]
```

**参数：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 起点星系 | 起始星系名称，支持模糊匹配 | 必填 |
| 终点星系 | 目标星系名称，支持模糊匹配 | 必填 |
| 安全模式 | super / capital / none | super |
| 跳跃距离 | 单次最大跳跃距离 (ly) | 6 |

**安全模式说明：**

| 模式 | 说明 |
|------|------|
| `super` / `s` / `安全` | 超级旗舰安全导航，避开所有危险星系 |
| `capital` / `c` / `旗舰` | 旗舰级安全导航 |
| `none` / `n` / `不限` | 无安全限制，所有星系均可途经 |

**示例：**

```
/nav 埃克瑟斯 出口
/nav 埃克瑟斯 出口 6 super
/nav e7-wsy ri-jb1 8 capital
/nav 埃克 出 10 none
```

### `/trinav` — 三神裔黑隐导航

```
/trinav <目的地星系>
```

**示例：**

```
/trinav 出口
/trinav ri-jb1
```

## 模糊匹配

星系名支持三种匹配策略（按优先级）：

| 策略 | 说明 |
|------|------|
| 精确匹配 | 名称完全一致 |
| 前缀匹配 | 前几个字符匹配，如 `埃克瑟` → `埃克瑟斯` |
| 子串匹配 | 名称包含关键字，如 `ri-jb1` → `RI-JB1` |

## 错误处理

| 场景 | 提示 |
|------|------|
| 终点为高安星系 (sec > 0.5) | 你确定这里能开诱导？ |
| 参数不足 | 显示命令用法 |
| 安全模式无效 | 参数错误提示 |
| API 请求超时 | 导航服务请求超时，请稍后再试 |
| API 返回错误 | 导航服务返回错误: HTTP xxx |
| API 找不到星系 | 返回 API 错误消息 |

## 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `NAV_API_BASE_URL` | 导航 API 地址 | `aio.dusy.run` |
| `NAV_API_TOKEN` | API 认证 Token | 内置默认值 |
| `NAV_MAX_JUMP_LY` | 默认最大跳跃距离 (ly) | `6` |

## 安装

1. 将本项目放入 AstrBot 插件目录
2. 启动 AstrBot，插件自动加载
